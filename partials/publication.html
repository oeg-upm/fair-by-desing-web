Online publication

    During this activity the ontology is made available online and accessible via its namespace URI as a machine-readable file and a human readable documentation using content negotiation.

Steps

1. Ontology uses an open protocol (HTTP/HTTPS)

                


                
                

                
2. Ontology is available in RDF

                

                    
                    
                

                

                

                


                
                

                
3. Ontology has content negotiation for RDF

                

                    
                    
                

                

                

                


                
                

                
4. Ontology has HTML documentation

                

                    
                    
                

                

                

                


                
                

                
5. Ontology URI is resolvable

                

                    
                    
                

                

                

                


                
                

                
6. Consistent ontology IDs are employed

                

                    
                    
                

                

                

                


                
                

                
7. A version IRI is declared in the ontology metadata

                

                    
                    
                

                

                

                


                
                

                
8. Ontology version IRI resolves

                

                    
                    
                

                

                

                


                
                

                
9. Ontology found in community registry

                

                    
                    
                

                

                

                


                
                

                
10. Ontology metadata are accessible, even when the ontology is not

                

                    
                    
                

                

                

                

                
            

        

    






    

    
Why?


    

        

        
    During this activity the ontology development team (possibly aided by system administrators) make the ontology available online. Since during this activity no new resources are generated, the guidelines correspond mostly to the technical aspects of the activity.
